# fstop-notifications

## Local Development

- Install docker
- Install NVM
- `nvm use 14`
- `docker-compose up` to run the DB locally
- Visit http://localhost:4000/
index.js to test
- `npm start` to start the core services
- `cd twitch-chat && npm start` to start the Twitch Chat integration
